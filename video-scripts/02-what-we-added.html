Video 2: What We Added (Paperless Ag)

Target: 4-6 minutes Type: Concept + Setup (standalone)

Before You Record

  • Paperless NGX running with documents loaded
  • Paperless Ag companion container running and connected
  • Claude Desktop installed but NOT yet configured with the MCP server (you'll do this live during recording)
  • Have your server URL and MCP auth token ready to paste (from the install script output or your .env file)
  • The keyword-fail demo uses "fertilizer recommendations" in Paperless search (returns zero results because the docs say "Nutrient Management Plan" and list products like "Anhydrous Ammonia 82-0-0"). The Claude demo asks the same thing and finds the right documents semantically.
  • Test the full flow once before recording so you know it works, then remove the MCP config so you can do it fresh on camera
The Problem with Keyword Search (0:00 - 0:40)
On Screen Narration
Show Paperless NGX dashboard, documents visible. "So we just looked at Paperless NGX — it's great for getting your paperwork digitized and searchable. But there's a catch with keyword search."
Click Type "fertilizer recommendations" into the Paperless search bar. Show zero or poor results. "Say I'm looking for fertilizer recommendations. I type that in and… nothing. But I know those documents are in here. The problem is the document is called a 'Nutrient Management Plan' and lists products like 'Anhydrous Ammonia.' Keyword search only works when you guess the exact words."
What Paperless Ag Adds (0:40 - 1:20)
On Screen Narration
Show Paperless dashboard still showing the failed search. Pause for a beat. "That's what Paperless Ag fixes. It's a small add-on that sits right alongside Paperless and gives it semantic search — meaning it understands what you're looking for, not just the exact words."
Show Keep Paperless visible. This is a talking point, no clicks needed. "It reads through every document you've uploaded and builds an understanding of what each one is about. All of that happens on your own machine — nothing gets sent to the cloud. Your documents stay private."
Connecting to Claude Desktop (1:20 - 3:00)
On Screen Narration
Click Switch to Claude Desktop. Show the main chat interface briefly. "The other thing we added is a connection to Claude — Anthropic's AI assistant. Instead of typing keywords into a search bar, you can just ask a question in plain English. Let me show you how to set that up."
Click In Claude Desktop, click Settings (gear icon or Claude menu → Settings). "Open up Settings in Claude Desktop."
Click Click "Developer" in the left sidebar, then click "Edit Config." This opens claude_desktop_config.json in your default text editor. "Go to Developer, then Edit Config. This opens up a settings file where we tell Claude how to talk to our Paperless server."
Type Paste the MCP config into the file. It should look like:

{ "mcpServers": { "paperless-ag": { "command": "npx", "args": ["-y", "mcp-remote", "YOUR_SERVER_URL/mcp", "--allow-http", "--header", "Authorization:Bearer YOUR_MCP_TOKEN"] } } }

Replace YOUR_SERVER_URL and YOUR_MCP_TOKEN with the real values from your install output. 		"Paste in this config. You need two things — your server's URL and the access token from when you set up Paperless Ag. The install script gives you both of these at the end. I'll paste mine in now."
Click Save the file and close the editor. "Save the file."
Click Quit Claude Desktop completely (Cmd+Q on Mac or close from system tray on Windows), then reopen it. "Now quit Claude Desktop completely and reopen it so it picks up the new config."
Click Open a new chat. Look for the MCP tools indicator — a small hammer/wrench icon near the text input, or a tools section showing "paperless-ag" is connected. "Open a new chat and look for the tools icon down here. You should see 'paperless-ag' listed — that means Claude is connected to your documents."
Try It Out (3:00 - 4:30)
On Screen Narration
Type Type into Claude: "What are my fertilizer recommendations?" "Let's try the exact same search that just failed. I'll ask Claude — 'What are my fertilizer recommendations?'"
Wait Wait for Claude to call the search_documents MCP tool and return results. It should find the Nutrient Management Plan documents and pull out the actual application rates and products. Let the response stream in fully. "It searched through all my documents and found the Nutrient Management Plan — even though I never said those words. It pulled out the actual products, the application rates, everything. Same question, completely different result."
Show Let Claude's full response be visible. Pause so viewers can read it. "That's the difference between keyword search and actually understanding what you're asking for. You don't have to remember the exact words in the document — just describe what you're looking for."
Wrap-Up (4:30 - 5:00)
On Screen Narration
Show Claude's response still visible with the search results. "So that's Paperless Ag — smarter search and AI integration for your documents, running right alongside Paperless NGX. Your Paperless stays completely stock, your data stays on your own hardware, and now you can just ask Claude to find what you need. In the next videos we'll cover what a VPS and Docker are, and then walk through setting up the whole thing from scratch."

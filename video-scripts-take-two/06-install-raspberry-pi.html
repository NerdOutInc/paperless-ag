Install on a Raspberry Pi

A home-network install for folks who want the archive on hardware they own instead of a monthly cloud server.

Target: 6-8 minutes Start: Pi hardware Finish: Paperless on local network

Plain narration: 06-install-raspberry-pi-transcript.txt

Before Recording

  • Use a Raspberry Pi 4 with 4 GB minimum, or a Pi 5. A USB SSD is strongly preferred for long-term database use.
  • Install Raspberry Pi Imager on the recording computer and have a blank SD card or SSD ready.
  • Prefer Ethernet for the demo. If using Wi-Fi, rehearse the Imager customization fields.
  • Have the same install command ready: curl -fsSL https://paperless.fullstack.ag/install.sh | bash
  • Plan edits for flashing, first boot, Docker pulls, and embedding catch-up.

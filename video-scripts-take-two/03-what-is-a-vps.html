Video 3

What Is a VPS?

Explain the rented-server idea before the viewer sees a real DigitalOcean install.

Target: about 2 minutes Start: VPS explainer page Finish: Droplet form, no create

Plain narration: 03-what-is-a-vps-transcript.txt

Before Recording

  • Use Helium and start on a public DigitalOcean VPS or Droplets page.
  • Keep the DigitalOcean session warm if showing the create Droplet form. Do not create a Droplet in this concept video.
  • If using the Paperless Ag deploy badge, open the site with open -a Helium during choreography so the address bar does not steal focus.

Choreography

Time On Screen Narration
0:00-0:25 Show DigitalOcean VPS or Droplets page, clean and already loaded. Hey this is Brian from Fullstack Ag. Before we install Paperless Ag in the cloud, it helps to know what a VPS is. VPS stands for virtual private server, but the plain version is simpler: it is a computer you rent online.
0:25-0:55 Show Slowly scroll a VPS explainer page or the Droplets overview. Instead of buying a physical computer, plugging it in at home, and keeping it online yourself, you rent a small slice of a server in a data center. The provider handles power, internet, cooling, and the hardware. You get a Linux machine that is always on.
0:55-1:25 Click Open the Paperless Ag deploy link or DigitalOcean Create Droplet form. Show size and region controls only. On DigitalOcean, a VPS is called a Droplet. When you create one, you pick a region, an operating system, and a size. For Paperless Ag, I like starting with at least 4 gigabytes of memory and 2 CPUs. That gives Paperless, Postgres, Redis, and the search add-on enough room to breathe.
1:25-1:55 Show Hold on the create form. Keep the pointer away from final create buttons and pricing footers. The nice part is that a VPS is already reachable from the internet, so you can open your document archive from the office, the house, or the field. The tradeoff is that it is a real server: use strong passwords, SSH keys, backups, and delete it when you no longer need it so you stop paying for it.
1:55-2:15 Show End on the Droplet form or the Paperless Ag deploy page. That is all a VPS is for our purposes: an always-on Linux computer rented from a cloud provider. In the install video, we will create one and put Paperless Ag on it step by step. Thanks for watching!

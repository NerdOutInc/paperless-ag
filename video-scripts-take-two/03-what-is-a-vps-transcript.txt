Before we install Paperless Ag in the cloud, 
it helps to know what a VPS is. 
VPS stands for virtual private server, 
and it's basically a computer you rent online.

You can install Paperless on a computer you have at home,
but installing it on a VPS is easy, and it gives you access
to your documents from anywhere.

On DigitalOcean, a VPS is called a Droplet.
When you create one, you pick a region,
an operating system, and a size.
For Paperless Ag, 
I like starting with at least 4 gigabytes of memory and 2 CPUs. 
That gives Paperless, Postgres, Redis, 
and our search add-on enough room to breathe.

The tradeoff to using a VPS over hosting it on a computer at home
is that it is a real server on ther internet:
use strong passwords, SSH keys, backups,
and delete it when you no longer need it so you stop paying for it.

That is all a VPS is for our purposes: 
an always-on Linux computer rented from a cloud provider. 
In the install video, 
we will create one and put Paperless Ag on it step by step.

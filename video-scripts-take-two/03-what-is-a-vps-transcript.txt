Before we install Paperless Ag in the cloud, it helps to know what a VPS is. VPS stands for virtual private server, but the plain version is simpler: it is a computer you rent online.

Instead of buying a physical computer, plugging it in at home, and keeping it online yourself, you rent a small slice of a server in a data center. The provider handles power, internet, cooling, and the hardware. You get a Linux machine that is always on.

On DigitalOcean, a VPS is called a Droplet. When you create one, you pick a region, an operating system, and a size. For Paperless Ag, I like starting with at least 4 gigabytes of memory and 2 CPUs. That gives Paperless, Postgres, Redis, and the search add-on enough room to breathe.

The nice part is that a VPS is already reachable from the internet, so you can open your document archive from the office, the house, or the field. The tradeoff is that it is a real server: use strong passwords, SSH keys, backups, and delete it when you no longer need it so you stop paying for it.

That is all a VPS is for our purposes: an always-on Linux computer rented from a cloud provider. In the install video, we will create one and put Paperless Ag on it step by step.
